# Go Alive – Event Management Web Application

## 📌 Project Overview

Go Alive is a web application designed to support youth communities in organizing and managing local and international events. The platform enables authorized users to add and manage event data through a secure and user-friendly interface.

This project was developed as part of an international initiative focused on youth engagement and cross-border collaboration.

---

## 🚀 Features

* 🔐 **Authentication System** – secure login system for authorized users to manage event content
* 📅 **Event Management** – create, edit, and manage event information for youth activities
* 📱 **Responsive Design** – fully responsive interface optimized for desktop and mobile devices
* 🎨 **UI/UX Implementation** – interface built based on a Figma design prototype
* ⚡ **Dynamic Data Handling** – real-time data interaction using JSON-based storage
